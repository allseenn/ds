### Задача кластеризации. Алгоритм К-средних

https://geekbrains.ru/tests/275/

Можно ли применять K-fold с алгоритмом K-means?

```
В применении k-fold нет смысла т.к. k-means - алгоритм обучения без учителя
```

Что такое центроид?

```
Среднее арифметическое всех элементов кластера
```

В чем отличается задача классификации от задачи кластеризации?

```
Кластеризация относится к разделу обучения без учителя, а классификация к обучению с учителем
```

Что происходит с целевой функцией во время обучения K-means?

```
Она уменьшается
```

Сколько центроидов создается в начале алгоритма K-means?

```
Столько же сколько и кластеров
```

Как подобрать оптимальное количество кластеров для алгоритма K-means? 

```
При помощи дендрограммы
При помощи диаграммы локтя
```
